## Description

This Age Calculator is a web application built using HTML, CSS, and JavaScript that allows users to input their birthdate and calculates their age in years, months, and days. It provides validation to ensure accurate inputs and displays error messages when necessary.

## Features

- Input fields for day, month, and year of birth.
- Real-time validation for valid input ranges (e.g., day: 1-31, month: 1-12, year: 1900-present year).
- Display of accurate age calculation.
- Responsive design for various screen sizes.

## Getting Started

### Prerequisites

- Web browser (Google Chrome, Mozilla Firefox, etc.)

### Installation

1. Clone or download the repository to your local machine.
2. Open the `index.html` file in a web browser.

## Usage

1. Open the web application in a web browser.
2. Enter your birthdate: day, month, and year.
3. Click the "Calculate Age" button.
4. Your age will be displayed below the input fields.
